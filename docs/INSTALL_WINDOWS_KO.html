Claude Stream Deck 플러그인

Windows 포맷 후 재설치 및 복구 안내서

설치 파일: com.hanbroz.claude-usage.streamDeckPlugin
이 파일을 더블클릭하면 Stream Deck이 설치 확인 창을 표시합니다. 별도의 MSI 또는 EXE는 필요하지 않습니다.

1. 포맷 전에 보관할 파일

  1. 이 ZIP 파일 전체를 USB, NAS 또는 클라우드 저장소에 보관합니다.
  2. Stream Deck 앱에서 설정 → Profiles → 아래쪽 화살표 → Backup All → Create Backup을 실행해 .StreamDeckProfilesBackup 파일을 별도로 보관합니다.
  3. Code Start에서 사용하는 프로젝트 폴더도 백업합니다. 드라이브 문자나 경로가 달라지면 복구 후 버튼 설정에서 새 경로를 지정해야 합니다.

프로필 백업에는 버튼 배치와 설정이 들어가지만 이 비공개 플러그인의 설치 파일은 포함되지 않습니다. 반드시 두 파일을 모두 보관하십시오.

2. Windows 포맷 후 설치 순서

  1. Elgato 공식 사이트에서 Stream Deck을 설치하고 한 번 실행합니다. 이 플러그인은 Stream Deck 7.1 이상이 필요합니다.
  2. Windows PowerShell에서 Claude Code를 설치합니다.
winget install Anthropic.ClaudeCode
claude --version
claude

처음 claude를 실행하면 브라우저 로그인 절차를 완료합니다. Native Windows에서는 Git for Windows 설치를 권장하지만, 없으면 Claude Code가 PowerShell을 사용합니다.

  1. com.hanbroz.claude-usage.streamDeckPlugin을 더블클릭하거나 Install.cmd를 실행합니다.
  2. Stream Deck의 설치 확인 창에서 설치를 승인합니다.
  3. 프로필 백업이 있으면 .StreamDeckProfilesBackup 파일을 더블클릭하고 Restore를 선택합니다. 기존 프로필을 덮어쓸 수 있으므로 새 설정을 만든 뒤에는 주의하십시오.
  4. 프로필 백업이 없으면 Claude 그룹의 버튼을 다시 배치하고 Code Start에 프로젝트명과 폴더를 입력합니다.
  5. Code Start 버튼을 한 번 누릅니다. 터미널이 열리고 Claude Code가 실행되며 상태 브리지가 자동 설치됩니다.

3. 정상 동작 확인

  • Code Start: 지정한 폴더의 터미널이 열리고 프로젝트명, 현재 모델, 컨텍스트 사용량 막대가 표시됩니다.
  • 세션 상태: 실행 중은 녹색, 답변 대기 중은 파란색, 종료 후에는 빨간색 Closed로 표시됩니다.
  • 버튼 이동: 실행 중 버튼을 다른 칸으로 이동해도 같은 세션의 모델과 컨텍스트 사용량이 이어집니다.
  • Usage 버튼: Claude Code에서 메시지를 한 번 처리한 뒤 5시간 및 7일 사용량이 표시됩니다.

4. 포맷 후 달라지는 항목

  • 포맷 전 실행 중이던 Claude 프로세스와 Code Start 캐시는 복구되지 않습니다. Code Start로 새 세션을 시작하면 다시 표시됩니다.
  • Claude 로그인 정보는 다시 인증해야 합니다.
  • 프로젝트 경로가 바뀌었다면 각 Code Start 버튼에서 폴더를 다시 선택하고 Save Settings를 누릅니다.

5. 문제 해결 및 롤백

  • 플러그인이 보이지 않으면 Stream Deck 앱을 완전히 종료한 뒤 다시 실행합니다.
  • 터미널이 열리지 않으면 PowerShell에서 Get-Command claude와 claude --version을 확인합니다.
  • 버튼이 STARTING에 머물면 프로젝트 폴더가 존재하는지 확인하고 버튼을 다시 누릅니다.
  • 설치 후 문제가 생기면 Stream Deck 설정에서 Claude 플러그인을 제거한 뒤 보관한 이전 버전의 .streamDeckPlugin을 다시 설치합니다.
